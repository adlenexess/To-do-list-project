# To-do-list-project
+
+A lightweight to-do list web app built with plain HTML, CSS, and JavaScript.
+
+## Implemented steps
+
+- Add new tasks.
+- Mark tasks as completed/uncompleted.
+- Edit existing tasks.
+- Delete tasks.
+- Filter tasks by All / Active / Completed.
+- Clear all completed tasks.
+- Persist tasks in `localStorage`.
+- Show remaining active task count.
+
+## Run locally
+
+Because this is a static app, you can open `index.html` directly in your browser or run:
+
+```bash
+python3 -m http.server 4173
+```
+
+Then visit `http://localhost:4173`.
